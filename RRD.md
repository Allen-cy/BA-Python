# RRD.md - 需求文档

## 1. 背景
将 Python 基础知识库（Markdown 格式）集成到 BA-Python 学习平台中，提供系统化的 17 阶段阶段式学习体验。

## 2. 核心需求
- **知识库集成**：解析本地 Markdown 知识库并转化为前端可用的 JSON 格式。
- **阶段式学习**：将 17 个知识模块映射为学习阶段，支持进度锁定与解锁。
- **AI 导师增强**：支持用户自定义提问，优先使用 Kimi-k2.5 模型提供辅导。
- **UI 交互优化**：在 Workspace 视图中增加持久化的提问框，并适配 17 阶段的 Dashboard 显示。

## 3. 业务目标
提升 BA 人士学习 Python 的系统性，通过 AI 导师解决实战过程中的即时问题。
