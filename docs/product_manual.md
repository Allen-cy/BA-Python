# 📖 BA-Python 用户手册

> 版本：v1.0.0 | 最后更新：2026-03-10

---

## 产品简介

BA-Python 是一个 Python 商业分析实战学习平台，帮助你通过真实业务场景，从零学习 Python 数据分析。

## 功能指南

### 1. 登录/注册

打开应用后，你将看到登录页面：

| 操作 | 说明 |
|------|------|
| **登录** | 输入邮箱和密码，点击「登录」 |
| **注册** | 点击「没有账号？立即注册」，填写显示名称、邮箱、密码 |
| **跳过登录** | 点击底部「跳过登录，体验演示版 →」可直接进入演示模式 |

> ⚠️ 演示模式下数据不会保存。如需保存学习进度，请注册账号。

### 2. 仪表盘（学习路线图）

登录后进入仪表盘，这里展示你的学习路线：

- **顶部进度条**：显示总学习进度
- **学习路线图**：展示 4 个阶段的课程
  - 🟢 已完成 — 绿色勾选
  - 🔵 进行中 — 蓝色播放按钮（脉冲动画）
  - 🔒 未解锁 — 灰色锁定
- **继续学习按钮**：一键跳转到当前进行中的课程

### 3. 学习工作台

点击「继续学习」或「继续当前课程」进入工作台：

#### 左侧面板：教程内容
- **业务背景**：真实商业场景描述
- **数据字典**：数据表字段说明
- **Python 提示**：折叠式提示，点击展开查看

#### 右上面板：代码编辑器
- 编写 Python 代码
- 点击「运行代码」执行
- 点击「提交挑战」提交答案

#### 右下面板：执行结果
- **控制台**：查看代码运行输出
- **数据视图**：预览 DataFrame 数据
- **可视化**：查看图表输出

### 4. 侧边栏导航

- **仪表盘** — 返回学习路线图
- **课程目录** — 浏览所有课程
- **社区问答** — 学员交流区（开发中）
- **个人中心** — 管理账号设置

### 5. 退出/返回

- 点击工作台左上角 Logo 返回仪表盘
- 点击侧边栏底部「退出」按钮登出

## 常见问题

**Q: 跳过登录后如何注册？**
A: 回到登录页面重新注册即可。

**Q: 忘记密码怎么办？**
A: 暂不支持密码重置，请联系管理员。

**Q: 代码运行没有反馈？**
A: 当前版本为模拟运行，真实 Python 执行功能将在 v1.1.0 上线。
