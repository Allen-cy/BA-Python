# 📑 BA-Python 需求文档 (RRD)

> 版本：v1.0.0 | 最后更新：2026-03-10

---

## 1. 项目背景

企业中的商业分析师越来越需要掌握 Python 数据分析能力。BA-Python 旨在提供一个沉浸式、实战化的学习环境，让用户在模拟真实业务场景中学习 Python。

## 2. 需求来源

- 用户痛点：传统 Python 教程脱离实际业务场景
- 市场机会：商业分析师岗位对 Python 技能需求持续增长
- 技术可行性：Supabase + Vercel + React 可快速构建全栈应用
