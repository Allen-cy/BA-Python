# 📋 BA-Python 产品需求文档 (PRD)

> 版本：v1.0.0 | 最后更新：2026-03-10

---

## 1. 产品概述

**BA-Python** 是一个基于真实业务场景的 Python 商业分析实战学习平台。面向零基础或初级商业分析师，通过分阶段的实战课程，帮助用户从零构建 Python 数据分析能力。

## 2. 目标用户

| 用户角色 | 描述 |
|----------|------|
| 初级商业分析师 | 具备基础业务知识，希望学习 Python 进行数据分析 |
| 运营/市场人员 | 需要用数据驱动决策的业务岗位 |
| 在校学生 | 商科/管理学专业，期望掌握数据技能 |
| 课程讲师 | 创建和管理课程内容（后期） |

## 3. 核心需求

### 3.1 用户认证
- **P0** 用户注册/登录（邮箱 + 密码）
- **P0** 会话持久化
- **P1** 演示模式（跳过登录体验）
- **P2** 第三方登录（GitHub/Google）

### 3.2 学习路线图
- **P0** 展示分阶段学习路线（4 阶段）
- **P0** 每个阶段显示标题、描述、时长、标签
- **P0** 阶段状态：已完成 / 进行中 / 未解锁
- **P1** 动态解锁逻辑（完成上一阶段后解锁下一阶段）

### 3.3 学习工作台
- **P0** 三面板布局：教程内容 | 代码编辑器 | 控制台输出
- **P0** 显示业务背景、数据字典、Python 提示
- **P0** 代码编辑（文本域）+ 行号显示
- **P1** 代码运行（集成 Pyodide）
- **P1** 代码提交 + 挑战验证

### 3.4 进度追踪
- **P0** 总学习进度百分比
- **P0** 每个课程的独立进度
- **P1** 学习时长统计

### 3.5 数据持久化
- **P0** Supabase PostgreSQL 数据存储
- **P0** 行级安全策略 (RLS)
- **P0** 自动 profile 创建

## 4. 技术约束

| 约束 | 说明 |
|------|------|
| 前端 | React 19 + TypeScript + Tailwind CSS 4 |
| 后端 | Vercel Serverless Functions |
| 数据库 | Supabase (PostgreSQL) |
| 部署 | Vercel |
| 认证 | Supabase Auth |

## 5. 非功能需求

- 首屏加载 < 3 秒
- 支持主流浏览器（Chrome, Firefox, Safari, Edge）
- 移动端适配（响应式布局）
- 数据传输加密 (HTTPS)
